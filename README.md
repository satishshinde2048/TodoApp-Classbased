### `Run Locally`

+ Clone the project:

      git clone https://github.com/satishshinde2048/TodoApp-Classbased.git


+ Navigate to the project directory:

      cd TodoApp-Classbased
                                                                                                       
+ Install dependencies:                                                                                                    

       npm install 
                                                                                                                                                                                                                
+ Start the server:                                                                                                        

       npm start
